## EECS 4443: Mobile User Interfaces - Lab 1
Simple login interface.

## Features
- **Three screen UI:**
  - **Login:** Where the user inputs credentials to access the Welcome screen.
  - **Register:** Where the user creates credentials to enable login.
  - **Welcome:** Where the user is redirected upon a successful login.
- Input validation to ensure all fields are completed and that credentials meet criteria.
- Option to show password and to remember user.
